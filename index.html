À propos

Métro Travel est un jeu quotidien, dont l'objectif est de trouver les différentes stations entre celle de départ et d'arrivée.

Le site est développé par Vincent Hardouin, le code source est disponible sur GitHub.

L'idée originale est de Marianne.

N'hésitez pas à me faire des retours, des suggestions d'améliorations et/ou à contribuer.

Uniquement les statistiques de visite sont récupérées.
